How to use

To ensure smooth operation and access to all features, such as downloading and searching models, it's essential to run the app using the command
`python app.py` . Without doing so, some functionalities may be unavailable. Running this command will activate the application, allowing you to fully explore its capabilities.

About

Web UI is a user-friendly, Open source web-based interface for interacting with Ollama, a tool for managing AI models and chat interactions. It is designed to resemble popular chat interfaces like ChatGPT, offering features such as model switching, chat history management , installing Ollama models.

  1. Ollama supports a wide variety of models—more than 120—available for download and usage.
  2. user-friendly
  3. Open source
  4. Beautiful UI
  5. Offline USE afer downloading Model
